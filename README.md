# Portfolio Dev
Projeto da Rocketseat para desenvolver um portfólio com HTML e CSS, com layout moderno e que apresenta sua identidade como desenvolvedor(a), suas principais habilidades e projetos em destaque. 
<br>
O portfólio conta com uma seção de apresentação personalizada, tags de tecnologias que identificam as stack e cards de projetos que exibem imagem, título e descrição.
